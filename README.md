# Como executar o projeto


## Front end web
Pré-requisitos: npm / yarn / pnpm

```bash
# clonar repositório
git clone https://github.com/RaulSAraujo/Front-MPS.git


# instalar dependências
## yarn
yarn

## npm
npm install

## pnpm
pnpm install

# executar o projeto
## yarn
yarn dev

## npm
npm run dev

## pnpm
pnpm dev
```
